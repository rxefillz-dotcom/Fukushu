# 今週やったことの復習（明日のテスト用・流れで）

明日の土曜テスト用に、**今週の流れ**で一通りおさらいします。

---

## 1. Git・GitHub の流れ

### やったこと
- Git を入れて、自分の名前を登録した
- リポジトリ（`my-learning-log`）を作って、変更を記録して GitHub に送った

### 覚えるコマンド（実行する順）

| 順番 | コマンド | 意味 |
|:----:|----------|------|
| 1 | `git add ファイル名` または `git add .` | 「このファイルを次の commit に含める」＝ステージング |
| 2 | `git commit -m "メッセージ"` | 変更を「記録する」 |
| 3 | `git push` | 記録を GitHub に送る |

**その他**
- **リポジトリを手元にコピーする:** `git clone https://github.com/ユーザー名/リポジトリ名.git`
- **名前を登録する:** `git config --global user.name "名前"`  
  - 穴埋めだと `user.name` の部分

### 流れのイメージ
```
ファイルを編集する
  → git add（含める）
  → git commit（記録する）
  → git push（GitHub に送る）
```

## 2. 木曜：HTML を「1枚」作った

### やったこと
- メモ帳 or Cursor で `index.html` を新規作成
- 中身を書いて保存し、ブラウザで開いて表示を確認した

### 書いたタグ（意味つきで覚える）

| 書いたもの | 意味 |
|------------|------|
| `<!DOCTYPE html>` | 「この文書は HTML5 です」の宣言 |
| `<html lang="ja">` | ページの言語＝日本語（`lang` は属性） |
| `<h1>～</h1>` | **いちばん大きい見出し** |
| `<p>～</p>` | **段落**（paragraph） |

### 流れのイメージ
```
index.html を新規作成
  → DOCTYPE・html・head・body の骨組みを書く
  → 見出しは <h1>、文は <p> で書く
  → 保存してブラウザで開く
```

## 3. 金曜：タグを増やして GitHub に上げた

### やったこと
- 木曜の `index.html` に「リスト」と「リンク」を足した
- `my-learning-log` フォルダに `index.html` を置いて、`git add` → `commit` → `push` した
- 用語（タグ・属性・セマンティクス）を README にメモした

### 足したタグ

| タグ | 読み方・意味 |
|------|----------------|
| `<h2>～</h2>` | 2番目に大きい見出し |
| `<ul>～</ul>` | **順序なしリストの全体**（u + 小文字の L。数字の 1 ではない） |
| `<li>～</li>` | リストの **1項目** |
| `<a href="URL">表示テキスト</a>` | **リンク**。飛び先の URL は **`href`** 属性に入れる |

### 用語（テストでも聞かれやすい）

| 用語 | 説明 |
|------|------|
| **タグ** | `<p>` や `<a>` のように、要素を表すマーク。`<タグ名>` で始まり `</タグ名>` で閉じる。 |
| **属性** | タグのオプション。例：`<a href="URL">` の **href**。リンクの飛び先は `href` に入れる。 |
| **セマンティクス** | 見た目より「意味」で書く考え方。見出しなら `<h1>`、段落なら `<p>` を使う。 |

### 流れのイメージ
```
木曜の index.html を開く
  → <h2>「今週やったこと」を追加
  → <ul> と <li> でリストを書く
  → <a href="..."> でリンクを書く
  → my-learning-log に index.html を置く
  → git add index.html → commit → push
  → README にタグ・属性・セマンティクスをメモして push
```

## 4. テストでよく聞かれること（一覧）

- **変更を記録する3つのコマンド（順番）:** ① `git add` ② `git commit -m "メッージ"` ③ `git push`
- **リポジトリをコピーするコマンド:** `git clone https://github.com/...`
- **名前を登録する穴埋め:** `git config --global user.name "名前"` → **user.name**
- **「次の commit に含める」コマンド:** `git add`
- **いちばん大きい見出しのタグ:** `<h1>`
- **段落のタグ:** `<p>`
- **リンクの飛び先を書く属性:** `href`（`<a href="URL">`）
- **順序なしリスト:** 全体＝`<ul>`、1項目＝`<li>`
