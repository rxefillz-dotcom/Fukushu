## 1. Git の復習（5分）

**やること:** 「変更を記録して GitHub に送る」流れを1回やる。

1. このファイル（`手を動かす復習_明日用.md`）を開く
2. 下の「今日のメモ」に「○月○日：手を動かす復習やった」と1行書いて保存
3. ターミナルで `復習` フォルダ（またはこのファイルがあるリポジトリのフォルダ）に移動
4. 次を順に実行：
   - `git add .` または `git add 手を動かす復習_明日用.md`
   - `git commit -m "手を動かす復習プランで復習した"`
   - `git push`

**覚えているか確認:**  
「次の commit に含める」＝`git add`、「記録する」＝`git commit`、「送る」＝`git push`

## 2. HTML の復習（10分）

**やること:** 1枚だけ HTML を新規で作って、今週覚えたタグを全部使う。

1. **どこか適当なフォルダ**（例：`context` や `復習` の下）に `review.html` という名前で新規ファイルを作る
2. 次のタグを**自分で書いて**（コピペじゃなくて）ページを完成させる：
   - `<!DOCTYPE html>`
   - `<html lang="ja">`
   - `<head>` と `<body>`
   - `<h1>` … ページのタイトル（例：「今週の復習」）
   - `<h2>` … 小見出しを1つ（例：「やったこと」）
   - `<p>` … 短文を1つ
   - `<ul>` と `<li>` … やったことリストを2〜3個
   - `<a href="URL">表示テキスト</a>` … 自分がよく見るサイトへのリンクを1つ
3. 保存して、**ブラウザで開いて**表示を確認する

**覚えているか確認:**  
リンクの飛び先はどの属性？ → **href**

## 3. Java の復習（15分）

**やること:** 今週やった3つ（Hello / 変数 / if・for）を、1本のファイルでまとめて書いて実行する。

1. **Java学習** フォルダに `Review.java` という新規ファイルを作る
2. クラス名は `Review` にする（`public class Review { ... }`）
3. `main` の中に、次の3つを**順番に**書く：

   **① Hello の代わりに**  
   `System.out.println("復習だよ");` を1行

   **② 変数**  
   `int score = 80;` と `String subject = "Java";` を用意して、  
   `System.out.println(subject + "の点数は" + score + "点");` で表示

   **③ if と for**  
   - `if (score >= 60)` なら「合格」、`else` なら「不合格」と表示
   - `for (int i = 0; i < 3; i++)` で「○回目」を3回表示（回数は `i + 1` でOK）

4. ターミナルでコンパイル＆実行：
   - `javac Review.java`
   - `java Review`

**覚えているか確認:**  
- 文の終わりには **`;`**  
- else の意味 → **「そうでなければ」**
